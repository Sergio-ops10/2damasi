# Documentación de Fine Tuning

Este repositorio contiene la documentación detallada paso a paso del proceso de fine-tuning de un modelo de IA (Qwen2.5-0.5B-Instruct) utilizando MLX en macOS.

## Contenido

- **index.html**: La página principal de la documentación.
- **script.js**: Contiene la lógica y los datos (títulos y descripciones) de cada paso.
- **style.css**: Estilos para el diseño "Premium" oscuro.
- **Imágenes**: Capturas de pantalla del proceso.

## Cómo ver la web

La forma más fácil de ver esta documentación es a través de **GitHub Pages**:

1. Ve a la pestaña **Settings** de este repositorio.
2. Haz clic en **Pages** en el menú lateral izquierdo.
3. En "Source", selecciona `Deploy from a branch` y elige la rama `main` (o `master`) y la carpeta `/ (root)`.
4. Guarda los cambios. GitHub te dará un enlace donde podrás ver la web funcionando con todas las imágenes.
