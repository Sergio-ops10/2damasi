Configuración inicial de la raspi y el mac mini

noviembre 25, 2025 Sergio López

 Documentación de Configuración Inicial

Este documento detalla los pasos de configuración inicial para la Raspberry Pi y el Mac mini.

 Configuración Inicial de la Raspberry Pi (Sistema Operativo Raspberry Pi OS)

Paso 1: Bienvenida al Escritorio Pi


Se muestra la pantalla de bienvenida al **Raspberry Pi Desktop**. Presiona **'Next'** para comenzar la configuración. Si se requiere, se solicita un reinicio para el auto-emparejamiento de dispositivos Bluetooth.

Paso 2: Conexión a la Red WiFi


Selecciona la red **'WiFi_AULA_ATECA'** de la lista y haz clic en **'Next'**.

Paso 3: Contraseña de WiFi


Introduce la contraseña de la red **'WiFi_AULA_ATECA'** y pulsa **'Next'** para conectar.

Paso 4: Actualización de Software (Verificación)


Se ofrece la opción de verificar y actualizar el sistema operativo y las aplicaciones. Pulsa **'Next'** para comenzar la comprobación y posible descarga/actualización, o **'Skip'** para continuar sin ella. Se descaraga TODO, esto puede tardar varios minutos, hay que tener paciencia

Paso 5: Crear USUARIO


Creamos usuario y contraseña que vayamos a usar para la raspi.

Paso 6: Elegir Navegador por Defecto


Selecciona el navegador web por defecto que deseas usar. En este caso se elige **Firefox**. También puedes marcar la opción para desinstalar el navegador no utilizado (Chromium).

 Configuración Inicial del Mac Mini (macOS)

Paso 1: Creación de Cuenta de Usuario


Se procede a la creación de una cuenta de usuario local. Se debe introducir el nombre completo, nombre de cuenta y la contraseña.

 

Paso 2: Escritorio Final (Mac Mini)


Una vez completada la configuración y el inicio de sesión, se muestra el escritorio de macOS, listo para su uso.
